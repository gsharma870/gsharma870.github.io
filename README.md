## 🚀 Hosting Options (Cheap & Reliable)

### Option 1: GitHub Pages — **FREE** ⭐ Recommended
Best for: Developers, free, easy updates via git
1. Create a GitHub account at github.com
2. Create a new repository named `yourusername.github.io`
3. Upload all files from this folder to the repository root
4. Your site will be live at `https://yourusername.github.io`

**Pros:** Free forever, custom domain support, very reliable
**Cons:** Requires basic git knowledge

---

### Option 2: Netlify — **FREE tier available** ⭐ Easiest
Best for: Non-developers, drag-and-drop deployment
1. Go to netlify.com and create a free account
2. Click "Add new site" → "Deploy manually"
3. **Drag and drop your entire `portfolio` folder** onto the upload area
4. Your site is instantly live on a `.netlify.app` URL
5. You can add a custom domain for ~$10–15/year

**Pros:** Free tier, instant deploys, drag-and-drop, great CDN
**Cost:** Free (with `.netlify.app` domain) or ~$19/month for pro features

---

### Option 3: Vercel — **FREE tier available**
Best for: Fast deploys, excellent performance
1. Go to vercel.com and sign in with GitHub
2. Import your repository or drag-and-drop
3. Instant deployment with global CDN

**Pros:** Extremely fast CDN, free tier is generous
**Cost:** Free for personal projects

---

### Option 4: Namecheap Hosting — ~$2–3/month
Best for: If you want a professional custom domain + hosting bundle
1. Buy domain + hosting at namecheap.com (~$2.99/month for hosting)
2. Upload files via cPanel File Manager or FTP
3. Domain suggestions: `gyanendrasharma.com` or `gsharma-research.com`
